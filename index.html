Model Details & Methodology

Model Architecture

  • Algorithm: XGBoost (eXtreme Gradient Boosting)
  • Objective: binary:logistic (for binary classification)
  • Feature Selection: Consensus of Boruta + LASSO + SVM-RFE
  • Cross-validation: 5-fold stratified CV with early stopping
  • Preprocessing: Recipes package for consistent feature engineering

Clinical Context

This model predicts the risk of postoperative hypokalemia in patients undergoing adrenal surgery. The four input features were identified through consensus feature selection across three machine learning methods.
